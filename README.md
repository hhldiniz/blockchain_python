#### About
  Application built with python that performs a election using blockchain.
#### Technologies
  The application is built with:
  * Python 3.6.5
#### Dependencies
  * Make sure Python 3.6+ is installed
#### Test

   ```yaml
   {  
    "1": "john",
    "2": "mary",
    "3": "george"
   }  
 * Run
    ```bash
    $ python _init__.py
    ```
